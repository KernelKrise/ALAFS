# ALAFS

Android LLM-driven Automated Fuzzing System

## Requirements

- Docker
- LLM Desktop of your choice ([Claude Desktop](https://claude.com/download) recommended)

## Build

```shell
./build.sh
```

> This will create MCP config file, you need to place it to your LLM Desktop environment.
> Example. To find you claude-desktop config: Settings->Developer->"Edit Config"
